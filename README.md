Pomodoro Focus Assistant Skill


一个基于扣子/Coze平台的AI生产力Agent，通过15分钟原子化工作法解决目标太多、注意力碎片化的问题，实现从大目标拆解到执行、复盘的完整闭环。


设计理念


这套系统的核心灵感来自于：


原子化任务：将模糊的大目标拆解为15分钟内可出明确结果的最小工作包，降低启动门槛
强制时间闭环：严格遵守15分钟工作+1分钟总结的节奏，避免陷入"完美主义"和"无限拖延"
反馈驱动迭代：不仅记录完成情况，还记录你的感受，根据实时反馈动态调整任务颗粒度
碎片化想法捕获：避免灵感流失，将随时产生的想法自动转化为可执行的任务


功能特性
🎯 目标拆解引擎


将大目标自动拆解为15分钟可完成的最小工作包
四个维度评分：易见效度、展示度、长期价值、紧急程度
支持多维度任务排序
⏱️ 15分钟专注循环


精准15分钟倒计时+系统通知
极简总结问题：1.你做了什么？2.你什么感觉？
你的回答原封不动记录，不修改、不润色
🔄 反馈驱动的任务更新


根据你的完成进度和感受，实时重新拆分大目标剩余部分
自动更新待办清单，动态调整任务颗粒度
下一个15分钟开始前，必发最新待办清单供你确认
💡 碎片化想法捕获


随时发送文字消息自动存入"想法池"
专注模式下静默收集：不打断，只存储不回复
跨会话统一存储：所有渠道想法汇总到同一文件
支持查看和清空想法池
📝 Obsidian日记自动导出


退出专注模式自动生成标准Markdown格式日记
可直接粘贴进Obsidian
自动携带标签：#专注日记 #每日复盘 #原子化执行
最后生成客观、有建设性的专注效果评价


快速开始
前置要求


一个扣子/Coze账号
已安装扣子/Coze应用（桌面端或移动端均可）
安装步骤


复制本项目中的 pomodoro_focus_skill_prompt.txt 文件内容
打开扣子/Coze，创建一个新的Bot/Skill
将复制的提示词粘贴到Bot的"系统提示词"或"人设"区域
保存并启用Bot
首次使用


发送 /goal 你的大目标（例如：/goal 搭建个人博客）
查看生成的15分钟工作包清单
发送 /focus 任务名称 开始第一个15分钟专注
15分钟后回答2个总结问题
查看更新后的待办清单，继续下一个15分钟
发送 /stop 退出并获取完整的Obsidian格式专注日记


使用指南
完整命令列表


命令	说明
/goal 大目标内容	将大目标拆解为15分钟工作包
/focus 任务名称	启动15分钟专注模式
/stop	退出专注模式，生成完整日记
/add 任务内容	添加任务到待办清单
/list	查看当前待办清单
/done 编号	标记对应任务为已完成
/priority 编号 1-5	设置任务优先级（1最高）
/sort easy	按易见效度排序
/sort show	按展示度排序
/sort long	按长期价值排序
/sort urgent	按紧急程度排序
/sort priority	按手动优先级排序
/clear	清空已完成任务
/ideas	查看想法池
/clearideas	清空想法池

| 编号 | 工作包内容 | 易见效度 | 展示度 | 长期价值 | 紧急程度 |

| ... | ... | ... | ... | ... | ... |

是否开始下一个 15 分钟？

我：/focus 查看 VitePress 官方部署文档

你：✅ 第 2 个 15 分钟专注开始｜任务：查看 VitePress 官方部署文档
